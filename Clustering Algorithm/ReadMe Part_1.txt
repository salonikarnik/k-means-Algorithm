To run the kMeans clustering algorithm for part 1:

Run as follows:

Navigate to the folder where the KMeans file is stored. In this case,it's stored in part1. From the part1 folder,execute the KMeans.Java File.

./KMeans <numberOfClusters> <input-file-name> <output-file-name>

Example:
./KMeans 3 D:/Machine Learning/Homework/Assignment 5/test_data.txt D:/Study/Machine Learning/Homework/Assignment 5/output.txt


The above code has been run for different values of k and following are the results of the program:

Note: If an error is thrown while running the program, re-run the program again. The error is generated due to the Random class function.
